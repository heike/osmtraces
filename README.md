# osmtraces

<!-- badges: start -->

<!-- badges: end -->

The goal of osmtraces is to help with downloading publicly available
traces from OpenStreetMaps

## Installation

You can install the development version of osmtraces like so:

``` r
remotes::install_github("heike/osmtraces")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(osmtraces)
## basic example code
```
